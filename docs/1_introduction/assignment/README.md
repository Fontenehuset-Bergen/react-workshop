# Uke 1: Oppgaver
Her er oppgavene for introduksjon til React, med vilje har jeg valgt at oppgavene ikke skal være nødvendig å løse i React siden det er fremdeles mye dere ikke har lært ennå. Istedet har jeg opprettet flere .ts filer som jeg ønsker at dere skal fylle ut. Når dere har fylt inn filene kan dere commite endringene til deres egen branch. Denne arbeidsmetoden vill være unik for denne uken, framover skal vi istedet gjøre alle oppgaver i React siden :)
