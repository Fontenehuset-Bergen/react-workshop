# Effecter

Denne uken skal vi lære om effekter i React, hvordan vi kan bruke det til å hente inn data og gjøre dette tilgjenglig for React. Her er det praktisk hvis dere er komfortabel med å skrive komponenter og bruke states før vi fortsetter på states, hvis dere trenger en oppfriskning er det bare å gå tilbake i dokumentasjonen og følge [README](../README.md) til riktige emner.
